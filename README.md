# 校园商家 AI 运营助手

这是“小满摘星计划 AI 顶尖人才实习项目”的笔试提交版 Demo。

项目用于验证：校园周边奶茶店、饭店、理发店、打印店等小微商家，是否需要一个低成本 AI 工具来生成活动文案、短视频脚本和顾客回复话术。

## 本地运行

本项目没有复杂依赖，不需要安装 Node.js。

1. 双击 `index.html`，打开 Demo 页面。
2. 双击 `product-plan.html`，打开一页 A4 产品验证方案。
3. 在浏览器里可以直接体验生成、复制和打印功能。

## 文件说明

- `index.html`：产品 Demo 主页面。
- `product-plan.html`：一页 A4 产品验证方案，适合打印成 PDF。
- `styles.css`：Demo 页面样式。
- `script.js`：Demo 交互和模拟 AI 输出逻辑。
- `README.md`：项目说明和提交说明。

## MVP 边界

当前版本不接入真实大模型 API，不做登录、数据库和后台管理。它的目的不是展示完整商业系统，而是用一个可运行的静态 Demo 验证产品思路。

## 笔试提交说明

必交内容：一页 A4 产品验证方案 PDF。

可选附件：Demo 页面链接、GitHub 仓库、演示视频。

本项目建议提交方式：

1. 将 `product-plan.html` 页面打印为 PDF。
2. 将整个项目上传 GitHub。
3. 如果有时间，部署到 Netlify / Vercel / GitHub Pages。
4. 在飞书表单中填写 PDF，并附上 Demo 链接。
